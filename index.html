Resumen de Galaxy 2.0: Software para PyMEs

Galaxy 2.0 es un software de gestión de escritorio para Windows, diseñado en Argentina, pensado para simplificar y potenciar la administración de pequeñas y medianas empresas (PyMEs). Una vez instalado, puedes acceder y trabajar desde cualquier dispositivo conectado a tu misma red Wi-Fi.

Nos enfocamos en la simplicidad: realiza compras, ventas, controla gastos y tu inventario sin complicaciones. Administra las cuentas corrientes de tus clientes y proveedores fácilmente. Todo en un lenguaje claro y sencillo para emprendedores.

¿Qué es Galaxy 2.0 y Cómo Funciona este Sistema de Gestión?

Galaxy 2.0 es una aplicación de escritorio diseñada para el sistema operativo Windows. Su objetivo es ser lo más simple e intuitiva posible, eliminando procesos complejos de otros ERP para que te concentres en lo importante de tu emprendimiento.

Una vez que instalas y ejecutas Galaxy 2.0 en tu computadora principal, el sistema se vuelve accesible desde cualquier otro dispositivo (como una tablet, notebook o smartphone) que esté conectado a la misma red Wi-Fi. Esto te brinda flexibilidad para consultar información o realizar gestiones desde diferentes puntos de tu negocio u hogar, sin necesidad de instalaciones adicionales en cada dispositivo.

El sistema se enfoca en las tareas esenciales de gestión: realizar compras, registrar ventas, controlar gastos y llevar un inventario actualizado de manera sencilla. También incluye un módulo de cuentas corrientes para que puedas evaluar el estado de cuenta con tus proveedores y el que tus clientes tienen contigo. Además, puedes personalizar fácilmente datos como el nombre de tu empresa, información de contacto y el signo de la moneda que utilizas en Argentina u otro país.

Módulos Principales del Software de Gestión

Clientes y Proveedores

Gestión completa de datos, contactos y sus respectivas cuentas corrientes. Ideal para el manejo de tu cartera.

Inventario y Productos

Control de insumos, productos simples y compuestos (recetas), stock, costos y precios de venta. Incluye gestión de unidades de medida y categorías para tu PyME.

Compras y Gastos

Registro y seguimiento de compras a proveedores y gastos generales del negocio de forma simple y eficiente.

Pedidos y Presupuestos (Ventas)

Creación y administración de presupuestos y pedidos de clientes, con seguimiento de estados y pagos. Optimiza tu proceso de ventas.

Pagos y Cuentas Corrientes

Registro detallado de pagos recibidos y realizados. Controla saldos pendientes de clientes y a proveedores.

Producción / Fabricación

Planificación y control de lotes de producción para productos compuestos, verificación de insumos y registro de unidades fabricadas. Software de producción integrado.

Dashboard y Reportes

Visualización de métricas clave del negocio. Exporta datos en formato XLSX para análisis detallado.

Mi Tienda (Base para Catálogo)

Módulo para visualizar productos y configuraciones, con posibilidad de listar precios para tus clientes.

Configuración General

Personaliza datos de tu empresa (nombre, CUIT, etc.), moneda, logo y más ajustes del sistema.

Galaxy 2.0 está diseñado y optimizado para funcionar en computadoras con sistema operativo Windows (versiones 7, 8, 10 y 11).

No, para el funcionamiento diario de Galaxy 2.0 no necesitas una conexión permanente a internet. Es una aplicación de escritorio que se instala y opera localmente en tu PC. La conexión a internet solo sería necesaria para descargar el instalador por primera vez o para futuras actualizaciones del software.

Una vez que Galaxy 2.0 está instalado y ejecutándose en tu computadora principal (servidor), puedes acceder a la aplicación desde otros dispositivos (como tablets, notebooks o smartphones) conectados a la misma red Wi-Fi local. Esto se hace ingresando la dirección IP local de tu PC principal seguida del puerto específico de Galaxy 2.0 en el navegador web del otro dispositivo (ej. http://192.168.1.100:8000). Encontrarás una guía detallada sobre cómo configurar esto en nuestra sección de Tutoriales.

La seguridad de tu información es una prioridad. Todos tus datos de Galaxy 2.0 se almacenan localmente en la base de datos dentro de tu propia computadora donde instalaste el software. Esto significa que tú tienes el control total sobre tu información y no se almacena en la nube ni en servidores de terceros. Recomendamos encarecidamente que realices copias de seguridad (backups) periódicas de tus datos para prevenir pérdidas ante cualquier eventualidad en tu equipo.
